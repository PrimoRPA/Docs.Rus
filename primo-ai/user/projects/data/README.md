# Данные

Данные — изображения документов, которые будут использоваться для обучения вашей модели. Из загруженных данных далее можно сформировать обучающий и тестовый датасеты для тренировки модели и для оценки ее способностей модели детектировать объекты на изображении. 

Изображения можно добавить в проект, просмотреть, редактировать, повернуть и удалить. Перечисленные действия осуществляются на странице **Данные**, которая доступна после перехода в выбранный проект.

![](<../../../../.gitbook/assets1/primo-ai/user-guide/data-project.gif>)
