# Установка компонентов сервера вручную

Порядок установки:
1. Установите Nginx.
1. Установите RabbitMQ.
1. Установите PostgreSQL.
1. Установите компоненты API:
   - установите Api;
   - установите Api.Auth;
   - установите Api.Inference;
   - установите Api.Logs;
   - установите Api.MachineInfo;
1. Установите UI.
