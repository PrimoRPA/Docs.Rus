# Общие сведения

Primo RPA AI Server — это сервис, который позволяет эффективно обрабатывать и распознавать данные из документов за счет использования искусственного интеллекта. AI Server включает готовые решения для типовых бизнес-задач, а также позволяет обучить нейросетевые модели под потребности конкретной организации. Модели AI Server могут использоваться как в роботизированных бизнес-процессах, так и в сторонних системах.


AI Server помогает организациям:
* Защитить конфиденциальные данные из документов за счет поддержки работы в закрытом контуре.
* Упростить процесс оцифровки документов.
* Классифицировать смешанные сканы документов по категориям.
* Проанализировать текст документа, чтобы извлечь ключевые данные или получить сжатое изложение.
* Автоматизировать маршрутизацию писем из общего ящика в конкретные отделы.
* Автоматизировать обработку резюме или заявок клиентов.



## Функциональные модули

В контексте решаемых задач можно выделить следующие модули:
* [Умный OCR](https://docs.primo-rpa.ru/primo-rpa/primo-rpa-ai-server/common/smart_ocr) — используется для обработки сканов документов с помощью сверточных моделей. Позволяет определять тип документа, распознавать данные в документах утвержденной или произвольной формы. Модуль включает набор обученных моделей от команды Primo RPA, а также предоставляет инструменты для обучения собственных моделей.
* [AI Текст](https://docs.primo-rpa.ru/primo-rpa/primo-rpa-ai-server/common/nlp) — используется для обработки естественного языка с помощью больших языковых моделей (LLM). Модели работают с информацией в текстовом виде, позволяют извлекать ключевые данные, суммировать выводы, классифицировать текст по категориям, генерировать новый текст по заданным параметрам.

## Архитектура

В общем виде AI Server представляет собой:
* Центр управления или AI Server, с которым пользователи работают через веб-интерфейс, а роботы или внешние приложения обращаются по API.
* Целевые машины с агентами, где обучаются и используются модели искусственного интеллекта. Каждый агент устанавливается на отдельной целевой машине.

Данная архитектура позволяет легко масштабировать решение. При необходимости одна модель может быть размещена на нескольких агентах, а нагрузка между ними будет распределяться сервером.


## Лицензирование

AI Server является коммерческим решением платформы Primo RPA и требует заключения лицензионного договора на использование. 

Лицензирование осуществляется без привязки к количеству обрабатываемых документов. Для использования продукта понадобится одна лицензия на AI Server и, как минимум, одна лицензия на агента. Количество лицензий агентов определяется сложностью выполняемых задач и инфраструктурой заказчика.
