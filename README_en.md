# General Information

Primo RPA Studio is a software development environment for creating automation scripts for business processes. The primary user of the application is an RPA developer.

The application is available in two editions: Enterprise and Community. The choice of edition depends on the intended use of automation.

![](<../../.gitbook/assets/editions-new-font.png>)

## Functionality

Regardless of the edition you choose, you will have access to the following key development features:
1. Creation of an RPA project - a collection of automation scripts.
2. Selection of a preferred script representation: sequence, diagram, or pure code.
3. Over 500 built-in components for interacting with third-party applications (desktop, web), databases, file systems, and other objects.
4. Additional libraries (dependencies) and the ability to create your own.
5. Quality assurance for the RPA script:
 * *automatic syntax checking of the code;*
 * *an integrated analyzer that assesses the quality of the script based on an internal set of rules.*
6. Step-by-step debugging of the script. Flexible configuration of the debugger and log recording, use of breakpoints, and a panel for monitoring variable states.
7. Importing projects from another RPA platform, such as Ui Path.
8. Creation of project templates for rapid development of new scripts.
9. Selection of interface theme and language.
