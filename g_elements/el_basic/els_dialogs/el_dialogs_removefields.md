![](<../../../.gitbook/assets/Удалить поля журнала.png>)

Позволяет удалить дополнительные поля из журнала Робота.

| Свойство | Тип                         | Описание        |
| -------- | --------------------------- | --------------- |
| Поля\*   | List<string>                | Массив полей, который нужно удалить из журнала |
