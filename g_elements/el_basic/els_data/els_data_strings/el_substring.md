Компонент, получающий подстроку из заданной строки.

| Свойство    | Тип    | Описание                |
| ----------- | ------ | ----------------------- |
| Строка\*    | String | Исходная строка         |
| Начало\*    | Int32  | Индекс начала подстроки |
| Длина       | Int32  | Длина подстроки         |
| Результат\* | String | Полученная подстрока    |
