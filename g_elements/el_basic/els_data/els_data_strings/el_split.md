Компонент, разделяющий строку на подстроки.

| Свойство      | Тип           | Описание                   |
| ------------- | ------------- | -------------------------- |
| Строка\*      | String        | Исходная строка            |
| Разделитель\* | String        | Разделитель строки         |
| Результат\*   | List\<String> | Массив полученных подстрок |
