Цикл, продолжающийся, пока установленное в нем выражение возвращает Истину (в начале цикла).

| Свойство    | Тип     | Описание                                                                |
| ----------- | ------- | ----------------------------------------------------------------------- |
| Выражение\* | Boolean | Выражение, истинность которого проверяется перед каждой итерацией цикла |
