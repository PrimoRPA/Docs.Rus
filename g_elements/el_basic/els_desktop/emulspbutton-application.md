# Приложение 1. Кнопки для эмулирования

Ниже приведены значения для свойств **Основная кнопка**, **Модификатор** и **Дополнительная кнопка** элемента [**Эмуляция спецкнопки**](https://docs.primo-rpa.ru/primo-rpa/g_elements/el_basic/els_desktop/el_desktop_emulspbutton). 

## Мышь

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| LBUTTON            | Левая кнопка мыши                  |
| RBUTTON            | Правая кнопка мыши                 |
| MBUTTON            | Cредняя кнопка мыши (или колесико) |
| ХBUTTON1           | Кнопка X1 - дополнительная четвертая кнопка мыши |
| ХBUTTON2           | Кнопка X2 - дополнительная пятая кнопка мыши |

## Клавиатура

### Клавиши функций

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| F1...F24           | Функциональные клавиши F1...F24 

### Клавиши для ввода данных (буквенно-цифровые)

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| VK_0...9           | Цифровые клавиши 0...9             |
| VK_A...Z           | Буквенные клавиши A...Z            |

| Кнопка             | Описание                           |

| SPACE              | Клавиша `Spacebar` (пробел)        |

| SELECT             | Клавиша `SELECT` (выбор). Может отсутствовать    |
| PRINT              | Клавиша `PRINT` (печать). Может отсутствовать    |

| ------------------ | ---------------------------------- |
| END                | Клавиша `End` (в конец текста)     |

| SEPARATOR          | Разделитель `\`               |
| DECIMAL            | Символ десятичного разделителя (`.` или `,`) |


### Прочие

| Кнопка             | Описание                           |

| MODECHANGE         | Запрос на изменение режима редактора метода ввода (IME) |

| PROCESSKEY         | Клавиша `IME PROCESS` (обработка IME) |
| CONVERT            | Преобразование в редакторе метода ввода (IME) |
| NONCONVERT         | Непреобразованный IME         |
| HANGEUL            | Режим Hanguel в редакторе метода ввода Windows (IME). Этот режим позволяет вводить корейские символы с помощью клавиатуры |
| KANJI              | Режим Kanji в редакторе метода ввода Windows (IME). Этот режим позволяет вводить японские иероглифы (кандзи) с помощью клавиатуры |
| JUNJA              | Режим JUNJA в редакторе метода ввода Windows (IME). В этом режиме пользователь может вводить кандзи, используя стандартные клавиши на клавиатуре, и редактор метода ввода Windows автоматически преобразует введенные символы в соответствующие кандзи |
| FINAL              | Последний режим редактора метода ввода (IME) |
