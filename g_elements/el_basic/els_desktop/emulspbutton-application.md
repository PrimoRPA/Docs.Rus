# Список кнопок для эмулирования

Свойствам **Основная кнопка**, **Модификатор** и **Дополнительная кнопка** соответствует список кнопок, приведенный ниже. 

Значения разделены на смысловые подгруппы.

## Кнопки мыши

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| LBUTTON            | Левая кнопка мыши                  |
| RBUTTON            | Правая кнопка мыши                 |
| MBUTTON            | Cредняя кнопка мыши (или колесико) |
| ХBUTTON1           | Четвертая кнопка мыши, боковая     |
| ХBUTTON2           | Пятая кнопка мыши, боковая         |

## Клавиши клавиатуры

### VK (Виртуальная клавиша?)

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| VK_(0-9)           | Эмуляция цифровых клавиш 0-9       |
| VK_(A-Z)           | Эмуляция буквенных клавиш A-Z      |

### F1-F24

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| F1...F24           | Эмуляция нажатия функциональных клавиш F1, F2...F24 |
