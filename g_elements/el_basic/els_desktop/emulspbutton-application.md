# Кнопки для эмулирования

[**Эмуляция спецкнопки**](https://docs.primo-rpa.ru/primo-rpa/g_elements/el_basic/els_desktop/el_desktop_emulspbutton) обладает свойствами **Основная кнопка**, **Модификатор** и **Дополнительная кнопка**. Значениям этих свойств соответствует список кнопок, приведенный ниже. Значения разделены на смысловые подгруппы.

## Мышь

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| LBUTTON            | Левая кнопка мыши                  |
| RBUTTON            | Правая кнопка мыши                 |
| MBUTTON            | Cредняя кнопка мыши (или колесико) |
| ХBUTTON1           | Четвертая кнопка мыши, боковая (X1) |
| ХBUTTON2           | Пятая кнопка мыши, боковая (X2)    |

## Клавиатура

### Клавиши для ввода данных

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| VK_0...9           | Цифровые клавиши 0...9       |
| VK_A...Z           | Буквенные клавиши A...Z      |

### Клавиши функций

| Кнопка             | Описание                           |
| ------------------ | ---------------------------------- |
| F1...F24           | Функциональные клавиши F1...F24 |
