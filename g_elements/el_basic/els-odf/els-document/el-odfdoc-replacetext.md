# Заменить текст

[](<../../../.gitbook/assets/image (128).png>)

Заменяет все вхождения исходного текста на новый.

| Свойство       | Тип    | Описание                 |
| -------------- | ------ | ------------------------ |
| Исходный текст | String | Текст, подлежащий замене |
| Новый текст\*  | String | Новый текст              |
