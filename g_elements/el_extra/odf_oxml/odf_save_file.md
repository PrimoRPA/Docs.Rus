# Сохранить документ

Компонент, производящий сохранения файла Word. Если путь к файлу не указан, сохранен будет файл, открытый в рамках текущего контейнера Word. 

## Свойства

1. **Путь к файлу** *[String]*: Путь к файлу Word (c:\folder\files.docx).
