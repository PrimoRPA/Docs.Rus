# Получить доступы файла

![](<../../../../.gitbook/assets/image (425).png>)

Элемент, получающий список доступов к файлу

| Свойство   | Тип                                         | Описание                                |

| ID файла   | String                                      | ID файла                                |

| Общий диск |                                             | Поддержка общих дисков                  |
