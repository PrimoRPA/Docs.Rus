# Стандартизация ФИО

![](<../../../.gitbook/assets/image (336).png>)

| --------- | --------------------------------------- | ---------------------------- |
| Токен     | String                                  | Токен API                    |

| Лимит     | int?                                    | Лимит ФИО                    |

| Json      | String                                  | Очищенное ФИО в формате JSON |
