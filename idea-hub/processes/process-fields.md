## Название

**Машинное имя**: `title`

**Тип поля**: Текст (простой)

**Описание**: Название процесса, которое указывается пользователем вручную при создании идеи или при редактировании процесса.


## ID процесса

**Машинное имя**: `field_id`

**Тип поля**: Текст (простой)

**Описание**: Уникальный идентификатор процесса, который автоматически присваивается сервером при создании идеи.


## Описание

**Машинное имя**: `body`

**Тип поля**: Текст (форматированный, длинный, с резюме)

**Описание**: Произвольное описание процесса, который пользователь указывает при создании идеи или при редактировании процесса.



## Департамент

**Машинное имя**: `field_area`

**Тип поля**: Entity reference

**Описание**: Ссылка на элемент управления, которому принадлежит процесс. Обязательное поле. 






## Ключ для расчёта

**Машинное имя**: `key`

**Тип поля**: Текст (простой)

**Описание**: Ключ используется для доступа к свойствам сущности в формулах.










## Автоматизация

**Машинное имя**: `field_automation_prc`

**Тип поля**: Число (десятичное)

**Описание**: Потенциальный процент автоматизации бизнес-процессов.



## Фактический процент автоматизации

**Машинное имя**: `field_automation_prc_real`

**Тип поля**: Формула

**Описание**: Фактический процент автоматизации бизнес-процессов.

**Формула**:

```
vars:
  total: |
    this.field_ops_total == 0 ? null : this.field_ops_total
  zero: this.field_ops_total - this.field_ops_exception
  result: |
    total == null ? total : zero == 0 ? 0 : (this.field_ops_success / (this.field_ops_total - this.field_ops_exception)) * 100
exp: result
```



### Отображение default
**Тип**: delta_formatter

**Настройки**: 
```
vars:
  value: avg(this.field('field_automation_prc_real'))
  delta: value - this.field_automation_prc
exp: |
  {"value": value, "delta": delta}
```

## Фактическая утилизация лицензии робота

**Машинное имя**: `field_disposal_licenses`

**Тип поля**: Формула

**Описание**: Количество лицензий робота в день, использованных для выполнения процесса.

**Формула**

```
(this.field_ops_total * this.field_ops_duration_avg) / 60 / 1440
```


### Отображение default
**Тип**: delta_formatter

**Настройки**: 
```
vars:
  value: avg(this.field('field_disposal_licenses')) * 100 * 30 / 30
  delta: (value - this.field_disposal_licenses_expected * 100 * 30 / 30)
exp: |
  {"value": value, "delta": delta}

```


### Отображение teaser
**Тип**: delta_formatter

**Настройки**: 
```
vars:
  orch: this.comp_orch_connected
  value: | 
    orch ? (avg(this.field('field_disposal_licenses')) * 30) : (this.field_disposal_licenses_expected * 30)
exp: |
  {"value": value}
```


## Плановая утилизация лицензии робота

**Машинное имя**: `field_disposal_licenses_expected`

**Тип поля**: Формула

**Описание**: Запланированное число лицензий робота, которое понадобится для выполнения процесса, из расчета в один день.

**Формула**

```
(this.field_ops_pd_expected * this.field_human_duration) / 1440 / 3
```


### Отображение default
**Тип**: delta_formatter

**Настройки**: 
```
{"value": this.field_disposal_licenses_expected * 100}
```

## Фактический FTE

**Машинное имя**: `field_fte`

**Тип поля**: Формула

**Описание**: Количество человеческого труда в текущий день, которое требуется для реализации процесса без автоматизации.

**Формула**

```
(this.field_ops_success * this.field_human_duration) / WORK_MONTH_MINUTES
```


### Отображение default
**Тип**: delta_formatter

**Настройки**: 
```
vars:
  value: avg(this.field('field_fte')) * 30
  delta: value - this.field_fte_savings
exp: |
  {"value": value, "delta": delta}
```


### Отображение teaser
**Тип**: delta_formatter

**Настройки**: 
```
vars:
  orch: this.comp_orch_connected
  prev: this.field_fte_savings
  value: | 
    orch ? (avg(this.field('field_fte')) * 30) : this.field_fte_savings
  delta: | 
    orch ? value - prev : null
exp: |
  {"value": value, "delta": delta}
```


## Неавтоматизируeмый человеческий труд

**Машинное имя**: `field_fte_after`

**Тип поля**: Формула

**Описание**: Количество человеческого труда, которое необходимо для реализации процесса с учётом автоматизации.

**Формула**:

```
((this.field_ops_error + this.field_ops_exception) * this.field_human_duration) / WORK_MONTH_MINUTES
```

### Отображение default
**Тип**: delta_formatter

**Настройки**: 
```
{"value": avg(this.field('field_fte_after')) * 30}
```

### Отображение teaser
**Тип**: delta_formatter

**Настройки**: 
```
vars:
  orch: this.comp_orch_connected
  value: | 
    orch ? (avg(this.field('field_fte_after')) * 30) : this.field_fte_savings * (100 - this.field_automation_prc) / 100
exp: |
  {"value": value}
```

## Плановый FTE

**Машинное имя**: `field_fte_savings`

**Тип поля**: Формула

**Описание**: Величина сокращения FTE за счёт применения робота, с учётом ошибок автоматизации и процента автоматизации. Другими словами, количество человеческого труда, которое больше не понадобится.

**Формула**:

```
((this.field_ops_pd_expected * this.field_human_duration) / WORK_DAY_MINUTES) * (this.field_automation_prc / 100)
```

### Отображение teaser
**Тип**: delta_formatter

**Настройки**: 
```
{"value": this.field_fte_savings}
```

## Статус

**Машинное имя**: `field_workflow`

**Тип поля**: Статус

**Описание**: Текущий статус жизненного цикла процесса.

По умолчанию доступны следующие статусы:
- **Идеация** (process_1_ideation) - процесс находится в стадии предложенной Идеи.
- **Проектирование** (process_2_design) - процесс находится на стадии проектирования.
- **Разработка** (process_3_development) - процесс находится в стадии разработки.
- **Тестирование** (process_4_testing) - процесс находится в стадии тестирования.
- **Развертывание** (process_5_deployment??) - процесс находится на стадии развертывания.
- **Эксплуатация** (process_6_production) - процесс находится в эксплуатации. В этом состоянии в процессе производятся расчёты эффективности.
- **Остановлен** (process_7_stopped) - процесс остановлен, не выполняет больше полезной нагрузки. Однако, данные по этому процессу всё ещё используются для расчёта эффективности в некоторых формулах.
- **Отклонён** (process_8_rejected) - процесс не принят к реализации.

Список значений может быть изменён.

Значение этого поля переключается на специальной странице, на которую можно попасть по ссылке рядом с названием статуса:
