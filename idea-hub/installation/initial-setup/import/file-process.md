# Файл process.xlsx

Файл `process.xlsx` позволяет массово добавить в Idea Hub информацию об автоматизированных бизнес-процессах. Данный файл заполняется вручную и затем загружается на странице **Импорт данных**. Заполненный файл должен называться `process.xlsx` — не допускается менять его название и расширение.

Скачать шаблон файла можно в веб-интерфейсе Idea Hub на странице **Импорт данных** по кнопке **Получить образец файла**. Данная страница доступна только пользователю с административными правами.

![](<../../../../idea-hub/resources/installation/import-get-sample-file.png>)

Шаблон содержит предустановленный набор полей процесса, который можно расширить в зависимости от потребностей организации. 

{% hint style="warning" %}
**Важно.** Добавление новых полей производится только в веб-интерфейсе Idea Hub — если вы добавите новые поля в файл шаблона, изменения не применятся. 
{% endhint %}

Таким образом, перед началом импорта процессов рекомендуется изучить шаблон `process.xlsx`. Если полей процесса недостаточно, добавьте их через веб-интерфейс согласно этой [инструкции](https://docs.primo-rpa.ru/primo-rpa/primo-rpa-idea-hub/processes/add-fields).

При изменении набора полей, шаблон `process.xlsx`, доступный для скачивания на странице **Импорт данных**, автоматически обновится. Скачивайте шаблон только после того, как все необходимые поля процесса добавлены.




## Описание полей файла process.xlsx

Данные вносятся в файл в соответствии с правилами и форматом, указанными в таблице ниже. Подсказки по заполнению также содержатся в самом шаблоне. Чтобы их увидеть, наведите курсор на название нужного поля.

**Минимальный набор полей, которые должны быть в наличии для создания полноценной карты процесса — id, title, dep, systems, auto_prc, transactions, hum_duration, hum_cost, launched, workflow.**


{% hint style="info" %}
**Примечание**. Возможность Idea Hub рассчитывать показатели зависит от наличия информация в полях данного файла. Хотя большинство полей не являются обязательными, если они не будут заполнены, многие показатели рассчитать будет невозможно. Чем меньше полей заполнено, тем меньше показателей Idea Hub может рассчитать. 
{% endhint %}

{% hint style="warning" %}
**Важно.** Для правильной обработки данных, представляющих собой календарные даты, ячейки таблицы, в которых сохраняются подобные данные, должны иметь формат "Дата".
{% endhint %}

{% hint style="warning" %}
**Важно.** Для корректной работы с файлом, не рекомендуется использовать в ячейках файла какие-либо формулы (за исключением полей effect_actual и effect_planned, в которых могут быть использованы формулы Idea Hub). Информацию по синтаксису формул в Idea Hub можно найти [здесь](https://docs.primo-rpa.ru/primo-rpa/primo-rpa-idea-hub/about/formulas).
{% endhint %}

## Импорт файла 

Заполненный файл `process.xlsx` следует загрузить в веб-интерфейсе Idea Hub на странице **Импорт данных**. Пользователю потребуется права администратора.

Для импорта данных:
* В строке **Процессы** нажмите на стрелочку рядом с действием **Импортировать**.
* Выберите пункт **Загрузить файл**.
* Выберите файл для загрузки и нажмите **Загрузить файл**.
* Затем нажмите кнопку **Импортировать**.
