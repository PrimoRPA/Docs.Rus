# Primo.AI.Server 1.0.11


# Обновления и улучшения

1. Добавили возможность загружать изображение при отправки NLP-запроса:
   * У элемента **Создать запрос NLP** появилось свойство **Изображение** с типом данных String. В значении свойства необходимо указывать путь до файла с изображением. Поддерживаемые форматы: JPG, JPEG, PNG.
   * В ключе маршрутизации элемента должен быть указан ключ `nlp-ocr`, чтобы сервер направил запрос мультимодальной модели. Другие типы модели не работают с изображениями.


Для работы с библиотекой Primo.AI.Server необходимо установить ее в Студию в качестве зависимости. Библиотеку можно скачать:
- С официального сайта [NuGet](https://www.nuget.org/packages/Primo.AI.Server.Linux).
- Напрямую из Студии с помощью [менеджера зависимостей](https://docs.primo-rpa.ru/primo-rpa/primo-studio/projects/manage-dependencies#menedzher-zavisimostei) (раздел Nuget.org).
